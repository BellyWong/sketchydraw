Windows and Panels

In addition to the document window, there are various panels and windows.

Object

Object creation is initiated by dragging the mouse. You can cut, copy and paste any object, even into other programs.

Command

By executing various commands from the main menu, then general image format or export or save the document. Modification of the text object is primarily done using the main menu.

Launch and Data file

You can save your picture either as a SketchyDraw file or as PortableDocumentFormat or TIFF,EPS,SVG. Choose Save To... and specify the type you wish (remember that although SketchyDraw can write PDF and TIFF,EPS,SVG, it can only read its own .skt3 format, so always save a copy in that format).
