Color Panel

Set the fill color and line color text color.

To display the Color panel:

  • "Tools" menu in the "Show Colors" command.

  • Or, in some "Inspector", click the colored square.

  • When using fill patterns and the colors at the bottom of the panel "Open pattern file..." click to select the image file pattern.

    The file format is TIFF, PNG, JPEG image format, such as a generic.

Using the pattern as line color, low-resolution images will be made even at a high resolution image files.
