Resize object

When you are in selecting mode, you can select any object:
And resize it by one of its knobs.
Or move it by dragging it around.

To scale an object:

  1. Tools panel "Select" button.

  2. Select the object to change the shape.

  3. Mouse button on the knob, move the mouse while dragging in the direction you want to transform.

  4. Release the mouse button.

    The shape of the object is fixed.

  5. You can zoom while maintaining the ratio.

    Option drag the mouse while pressing on the knob.

  6. You can also zoom using the inspector.

    "Dimension" "Dimensions" can be scaled by changing the width or height values.
