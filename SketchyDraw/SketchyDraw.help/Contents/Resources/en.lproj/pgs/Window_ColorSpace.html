ColorSpace Panel

Set the color space to use.

To view a color space panel:

  • "Tools" menu in the panel "ColorSpace..." Command.

    /System/Library/ColorSync, /Library/ ColorSync, ~/Library/ColorSync ICCProfile placed in each of the lists for more information.ICCProfile not be applied in advance, please keep to install one of these folders.

  • If you set the model for different models and fill color, line color, those colors are returned when the color space and color settings.
    Usually "generic RGB" or "Generic CMYK" and returned.

  • When you save a document, the name will also be saved by applying the color space.

When applied to text object is not obtained does not apply because the practical processing speed.
