Moving Objects

Change the position of the object once created.

To move an object:

  1. Select the object moves.

  2. Drag the mouse button is displayed inside the knob.

    Object moves along well.

  3. Release the mouse button.

    Moving object is complete.

  4. You can also navigate using "Inspector".

    "Dimension" "Dimensions" of X or Y can be moved by changing the value.

  5. You can also rotate using the inspector.

    "Dimension" "Rotate" can be rotated using a circular knob.

If two or more selected objects, drag the mouse on an object inside the knobs of any one.
When the mesh grid is enabled, allows to engage the upper left position of the grid position.
When the mesh grid is enabled, and if you move the arrow keys to move the grid size.
