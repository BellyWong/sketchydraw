Preferences

Set the default settings when creating a new document and a general execution environment.

View or change your preferences:

  • Main Menu "Sketchy Draw" "Preferences..." Preferences panel appears when you run the command.


For general preferences:

  • "General" tab and click the mouse

  • "Antialias" is set antialiasing display whether the object.
    Drawing upon the quality of the image object and the print quality is affected.

  • "High quality image screen" sets the rendering quality of the image object.
    Not affect the printer.

  • "Background" sets the background color of the document window.
    The panel appears when you click the border color around the color wells.

  • "Measurement unit" specifies the number of units when the primary inspector on a variety of settings.

  • "Options for TIFF document" the document sets the quality when exporting TIFF format.

  • "Autosave documents" to enable the automatic saving feature works when editing the document.
    Files in the folder that the document is saved (Autosaved) will be created automatically appended to the file name.


To set default values:

  • "Default" tab and click the mouse

  • "Default Text Font" is created when you create a new document and make first TextObject specify the font to be set. "Set... "Please specify the Font panel to display at the click of a button.Color well left the button, specify the font color.The panel appears when you click the border color around the color wells.

  • "Default Pagelayout" sets the paper size is set when you create a new document. "Set... "To display the Page Setup dialog box, click the button to set the paper size.

  • Bottom of the table is intended to specify the default color space when you create a new document.

To change the color space settings for each individual document, the main menu "Tools" panel "ColorSpace..."Command to set the panel to display the color space.
