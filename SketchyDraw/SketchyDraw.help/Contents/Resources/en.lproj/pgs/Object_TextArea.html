To create a text object:

  2. Move the cursor over the document window.

    The cursor changes to a + beam.

  3. Drag the mouse to move the mouse cursor where you want to add text.

    Rectangle is drawn and release the mouse button determines the length of the line of being.The cursor changes to a vertical rod, allowing you to enter characters from that location.

    When the new line, Return or the Enter key to enter.
    Each time a new line, grows increasingly vertical text objects.

  4. Go to objects outside the text editing cursor will create another new text objects and drag the mouse.
