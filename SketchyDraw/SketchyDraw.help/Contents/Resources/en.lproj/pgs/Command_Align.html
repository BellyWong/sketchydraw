Align and resize

Made the same size and alignment or the alignment of multiple objects, or make natural size an object.

To arrange the placement of multiple objects:

  1. Select multiple objects to align.

  2. Main menu "Format", "Align and Size" command in the appropriate.

    "Left Edges", "Right Edges", "Top Edges", "Bottom Edges," "Horizontal Centers", "Vertical Centers", "Equal arrangement" is the appropriate command, respectively.
    In "Equal arrangement", according to the display sequence foreword of an object, the bottom is arranged from a top from the left to the right.


You made the same size of multiple objects:

  1. Select multiple objects to align.

  2. Main menu "Format", "Align and Size" command in the appropriate.

    "Make Same Width" and "Make Same Height" and the appropriate command.


To make natura size an object:

  1. Choose one of the object that make natural size.

  2. Main menu "Format", "Align and Size", "Make Natural Size" command.

    If the text object, height and width are adjusted by the minimum value.
    In the case of a picture object, it is returned at the original size of the picture.
    In the case of other objects will be adjusted to the size of the square.
