Select Objects

To correct the picture, you must first select the object to modify.

To select an object:

  1. Tools panel "Select" button.

  2. Then move the cursor over the document window.

    Arrow cursor changes to a.

  3. Click the mouse cursor to select the location of the object.

    8 knobs appear around the object is selected.


To select multiple objects:

  1. Tool Panel "Select" button.

  2. Shift-click multiple objects sequentially while pressing.

  3. Or draw a rectangle by dragging the mouse.

    More than one object is selected will be present in the square.

  4. "Edit" in the menu "Select All" you can select the whole object of the command.


To cancel the selection:

  1. Tool Panel "Select" button.

  2. When you click on another object, the knob is visible around the object, the object before the knob off.

  3. When multiple objects are selected, Shift key while you click an object.

  4. "Edit" in the menu, "you do not select anything," you can unselect the whole object has been selected by the command.

Modifications are reflected in the object is always marked with the knob.This knob also appears immediately after creating a new object.
