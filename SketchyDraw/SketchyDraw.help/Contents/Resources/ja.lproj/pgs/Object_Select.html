オブジェクトの選択

図を修正するには、最初に修正するオブジェクトを選択する必要があります。

オブジェクトを選択するには：

  1. ツールパネルの「選択」ボタンをクリックします。

  2. 次に、ドキュメントウィンドウ上にマウスカーソルを移動します。

    カーソルが矢印の形に変わります。

  3. 選択するオブジェクトの位置にマウスカーソルを移動してクリックします。

    そのオブジェクトが選択され周囲に8個のノブが表示されます。


複数のオブジェクトを選択するには：

  1. ツールパネルの「選択」ボタンをクリックします。

  2. Shiftキーを押しながら複数のオブジェクトを順次クリックします。

  3. または、マウスドラッグにより矩形を描きます。

    矩形内に存在する複数のオブジェクトが選択状態になります。

  4. 「編集」メニュー内の「すべてを選択」コマンドの実行により総てのオブジェクトを選択することもできます。


選択を解除するには：

  1. ツールパネルの「選択」ボタンをクリックします。

  2. 別のオブジェクトをクリックすると、そのオブジェクトの周りにノブが表示されて、前のオブジェクトのノブは消えます。

  3. 複数のオブジェクトが選択されているときは、Shiftキーを押しながらオブジェクトをクリックします。

  4. 「編集」メニュー内の「なにも選択しない」コマンドの実行により選択されている総てのオブジェクトを非選択にすることができます。

修正は常にノブがついているオブジェクトに反映されます。オブジェクトを新規に作成した直後もこのノブが表示されます。
