テキストオブジェクト

文字列のオブジェクトです。

テキストオブジェクトを作成するには：

  1. ツールパネルの「テキスト」ボタンをクリックします。

  2. ドキュメントウィンドウ上にマウスカーソルを移動します。

    カーソルが+ビームの形に変わります。

  3. テキストの追加をしたい場所へマウスカーソルを移動してマウスドラッグします。

    マウスボタンを離すと矩形が描かれ、とりあえずの1行分の長さが決まります。カーソルが縦棒状に変わり、その場所から文字を入力することができるようになります。

    改行をするときは、Return又はEnterキーの入力をします。
    改行される度に、テキストオブジェクトがだんだん縦に大きくなっていきます。

  4. マウスカーソルを編集中のテキストオブジェクトの範囲外に移動して、マウスドラッグをすると別のテキストオブジェクトが新規に作成されます。


テキストオブジェクトを修飾するには：

  1. メインメニュー「ツール」「インスペタターパネルを表示」コマンドを選択します。

  2. テキストオブジェクへマウスの位置を合わせマウスクリックしてオブジェクトを選択します。

  3. 「インスペタターパネル」「線と塗り潰し」の「線」を有効にするとアウトラインの線幅を指定できます。「塗り潰し」を設定するとテキスト領域の背景を指定した色で塗り潰します。

    「線」の「線幅」が0.0のとき中抜き文字になります。

  4. 「線と塗り潰し」が「なし」でないときに、「インスペタターパネル」「破線と影」の「影」を設定するとテキスト領域に影がつきます。


文字を修飾するには：

  1. フォントやサイズを設定するときは、メインメニューの「フォーマット」から必要なコマンドを実行します。

  2. フォントを設定するときは、「フォーマット」「フォント」「フォントパネルを表示」コマンドを実行してフォントパネルを表示させます。

  3. カラーを設定するときは「ツール」「カラーパネルを表示...」コマンドを実行してカラーパネルを表示させます。

    またはフォントパネルからでもカラーを設定することができます。

  4. カーニング、リガチャ、アンダーライン、シャドーその他の設定は、それぞれのコマンドを実行します。

  5. テキストの「揃え」などを設定するときは、「テキスト」「中央揃え」などのコマンドを実行します。

  6. 「テキスト」「表」コマンドを実行すると、表組もできます。ただし、「インスペクターパネル」「線と塗り潰し」が「なし」でない場合には罫線が消えます。

    罫線有りの背景色有りにする場合には、「テキスト」「表」コマンドから表示されるダイアログの「セルの背景」を「カラーで塗り潰す」とします。ただしパターンカラーを使ってもそのカラーを保存することはできません。真っ黒になります。


テキストを編集するには：

  1. ツールパネルの「選択」ボタンをクリックします。

  2. 編集するテキストオブジェクへマウスの位置を合わせマウスクリックしてオブジェクトを選択します。

  3. マウスでダブルクリックをして文字を選択します。

  4. 文字の削除・置き換えなどを実行します。

  5. 挿入は選択を解除して、カーソルを縦棒状にし、カーソルの位置へ挿入する文字を入力します。
