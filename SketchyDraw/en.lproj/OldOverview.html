The Inspector Panel is used to change various attributes of objects such as whether it is filled, what gray and linewidth to use, whether a line has an arrow on it, etc.

The Control key will temporarily put you in Select mode.
The Shift key allows you to add or remove objects from the current selection.
Alternate allows you to constrain a resize or creation (e.g. constraining the creation of a Polygon keeps it from closing itself up, a rectangle will be a square, a PDF picture will keep its proper aspect ratio).

You can save your picture either as a SketchyDraw file or as PortableDocumentFormat or TIFF,EPS,SVG. Choose Save To... and specify the type you wish (remember that although SketchyDraw can write PDF and TIFF,EPS,SVG, it can only read its own .sktd format, so always save a copy in that format).
